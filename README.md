# Wanderlust

 ## Tech Stack

**Frontend:** EJS, Bootstrap

**Server:** Node JS, Express JS

**Database:** Mongo DB

## Demo
https://wanderlust-bdeg.onrender.com

## Run Locally

Clone the project

```bash
  git clone https://github.com/Suman-sadhu-khan/Wanderlust.git
```

Go to the project directory

```bash
  cd Wanderlust
```

Install dependencies

```bash
  npm install
```
